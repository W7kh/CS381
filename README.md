## 📋 Project Overview

This is a fully responsive, semantic, and accessible personal academic portfolio website built from scratch using only HTML5 and custom CSS3. The website showcases my academic achievements, technical skills, projects, and contact information.

## ✨ Features

### Technical Implementation
- ✅ **Semantic HTML5**: Proper use of semantic elements (header, nav, main, section, article, footer)
- ✅ **Custom CSS3**: No frameworks or templates - all styling written from scratch
- ✅ **Mobile-First Design**: Responsive layout that works on all devices
- ✅ **Flexbox Layouts**: Navigation bar, hero section, cards, and content alignment
- ✅ **CSS Grid Layouts**: Skills section, projects grid, achievements grid
- ✅ **Web Accessibility**: ARIA labels, skip links, keyboard navigation, semantic structure
- ✅ **No JavaScript**: Pure HTML/CSS implementation

### Pages
1. **Home (index.html)** - Overview with hero section, about, skills, and featured projects
2. **Profile (profile.html)** - Detailed education, projects, achievements, and skills breakdown
3. **Contact (contact.html)** - Contact information, form, and availability details

## 🎨 Design Features

### Responsive Breakpoints
- **Mobile**: < 576px (single column layout)
- **Tablet**: 576px - 991px (2-column grid)
- **Desktop**: ≥ 992px (multi-column grid, horizontal layouts)

### Color Scheme
- Primary: #3498db (Blue)
- Secondary: #667eea → #764ba2 (Gradient Purple)
- Dark: #2c3e50 (Navy)
- Light: #ecf0f1 (Light Gray)
- Success: #27ae60 (Green)

### Layout Techniques
- **Flexbox**: Navigation, hero content, cards, footer
- **CSS Grid**: Skills, projects, achievements, availability sections
- **Mobile-First**: Base styles for mobile, progressively enhanced for larger screens

## ♿ Accessibility Features

1. **Skip to Content Link** - Keyboard users can skip navigation
2. **ARIA Labels** - Proper labeling for screen readers
3. **Semantic HTML** - Meaningful structure for assistive technologies
4. **Focus Indicators** - Clear visual feedback for keyboard navigation
5. **Alt Text** - Descriptive text for images (when used)
6. **Color Contrast** - WCAG compliant color combinations
7. **Form Labels** - All form inputs properly labeled
